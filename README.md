## Installatie
#### This project is made with
* Node.js
* Express
* EJS

#### Clone the repository
```
git clone https://github.com/kiara1404/so-nuts
```

#### Installeer alle packages
```
npm install
```

#### Start de applicatie lokaal
```
nodemon run dev
```
